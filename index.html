PROJECTS

Kanban App

This Kanban app is a web-based task management tool that helps you organize, track, and manage tasks visually using the Kanban methodology. Tasks are displayed in columns representing different stages of a workflow, making it easy to see what needs to be done, what’s in progress, and what’s completed.

Podcast App

The Podcast App is a feature-rich podcast browsing and listening platform. Users can search shows, filter by genre, sort by various criteria, explore recommended shows via a carousel, view detailed podcast seasons and episodes, listen through a global audio player, and save favourite episodes — all with a responsive, theme-toggle UI.
